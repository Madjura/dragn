class allsteps.allsteps.FakeAlias(identifier)¶

Bases: object

Used for running dragn without starting the Django server. Required because the Alias is stored in the database and not accessible properly without the server. This is used as a replacement and functions exactly as the normal Alias would, except it does not come from the database, neither is it stored there.

save()¶

Blank method to bypass the Django database regarding the Alias System.
